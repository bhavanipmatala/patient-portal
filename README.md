# DemoPatientPortal Application

This project contains the complete source code for the DemoPatientPortal, a web application for patient-provider communication.

## Project Structure

The project is divided into three main parts:

- **/frontend**: Contains the React + TypeScript frontend application.
- **/backend**: Contains the Node.js + Express + TypeScript backend API.
- **/database**: Contains the SQL script for setting up the Microsoft SQL Server database.

Each directory has its own `README.md` file with specific setup instructions. Please follow the instructions in the order below.

## Setup Instructions

To set up and run this project on your local machine, please follow these steps in order:

1.  **Database Setup**: Navigate to the `database` directory and follow the instructions in its `README.md` to set up your SQL Server database.
2.  **Backend Setup**: Navigate to the `backend` directory and follow the instructions in its `README.md` to configure and run the backend server.
3.  **Frontend Setup**: Navigate to the `frontend` directory and follow the instructions in its `README.md` to configure and run the frontend application.

After completing these steps, the application will be running on your local machine.
